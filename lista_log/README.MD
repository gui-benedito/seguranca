# Lista de Logs - Scripts

Este diretório contém scripts em Bash para resolução de exercícios relacionados a análise de logs no Linux.

---

## Estrutura

Cada arquivo representa uma atividade:

```
atv_1
atv_2
atv_3
...
```

- Cada script é independente
- Cada um resolve um exercício específico

---

## Execução

Dar permissão para o script:

```bash
chmod +x atv_X
```

Executar:

```bash
./atv_X
```

Substitua `X` pelo número da atividade.

---

## Observações

- Alguns scripts podem precisar de sudo
- Os scripts utilizam logs do sistema (ex: /var/log)
- Os resultados são exibidos diretamente no terminal
